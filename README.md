# Netflix Style Landing Page

A responsive, Netflix-inspired landing page built with HTML and CSS. This project demonstrates modern web design principles with a focus on responsive layout, navigation, and content presentation.

## Features

- **Responsive Design**: Mobile-first approach with smooth adaptation to larger screens
- **Fixed Navigation Header**: Sticky header with Netflix logo and navigation menu
- **Hero Section**: Eye-catching welcome section with call-to-action buttons
- **Trending Content**: Showcase of trending movies/shows with poster images
- **New Releases**: Display of new content releases
- **Continue Watching**: Section for resumed content
- **User Profile**: Profile picture display in header
- **Smooth Scrolling**: Navigation links with smooth scroll behavior

## Sections

### Header
- Netflix logo
- Navigation menu (Home, Trending, New, Contact)

### Hero Section
- Main heading: "Watch something memorable."
- Tagline: "Stream movies, series and more — all in one place."
- Call-to-action buttons: "Watch Now" and "Browse"

### Trending Now

### New Releases
- Display of newly released content

### Continue Watching

## Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Advanced styling and layout
  - Flexbox for layout
  - CSS Grid for content sections
  - Media queries for responsive design
  - CSS variables for theming

## Color Scheme

- Background: `#0b0b0b` (Deep black)
- Text: `#ffffff` (White)
- Secondary Text: `#b3b3b3` (Light gray)

## How to Use

1. Open `index.html` in a web browser
2. Navigate through sections using the header menu or scroll
3. Click "Watch Now" or "Browse" buttons for navigation
4. Responsive layout adapts automatically to screen size
